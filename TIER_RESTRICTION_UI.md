## Color Scheme

- Warning color: `warning.main` (typically orange/yellow)
- Warning chips: `color="warning"` with `variant="outlined"`

- Icons: `<WarningIcon color="warning" />`

## Interaction

All restriction indicators are clickable and navigate to `/admin/subscription` page where users can:
1. View their current tier
2. See all restricted features
3. Upgrade to a higher tier
4. Compare tier features
